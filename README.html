# Dados_pessoais Esta aplicacao consiste em enviar dados de campos preenchidos pelos utilizadores, que sao depois enviados para outra janela, com os campos respetivos a cada dado inserido. A proposta de aplicacao acrescentei algumas formatacoes para diferenciar a parte da string afeta ao dado e o dado inserido pelo utilizador, bem como algumas validacoes e respetivas mensagens de erro.
